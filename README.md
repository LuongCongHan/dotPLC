# dotPLC
A .NET Library for Mitsubishi Connectivity, Modbus RTU, Modbus TCP.

## Overview
dotPLC is a library to support communication with Mitsubishi PLC, GX Simulator3, Modbus device, includes:
| Protocol        | Object          |
|-----------------|-----------------|
| SLMP for Client | SLMPClient      |
| SLMP for Server | SLMPServer      |
| GX Simulator3   | Simulator3      |
| Modbus RTU      | ModbusRtuClient |
| Modbus TCP/IP   | ModbusTcpClient |
